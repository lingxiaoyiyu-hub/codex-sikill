# 标题模板库

核心思路：标题不是照抄题材，而是复用结构。优先使用时间、冲突、数字、反转、利益、提醒、争议、人物处境。

## 通用模板

### 数字具体化型

`【事件】之后，真正值得看的不是【表面结果】，而是这【数字】个细节`

适合：热点复盘、体育赛后、政策变化、社会争议、平台规则。

### 时间节点型

`【时间】之后，【人物/机构】突然【动作】，这件事可能没那么简单`

适合：连续热点、突发进展、舆论反转、官方回应、后续处理。

### 冲突动作型

`【人物/机构】刚【动作A】，【另一方】就【动作B】，真正的焦点其实是【核心矛盾】`

适合：国际事件、商业竞争、职场争议、体育冲突、平台变化。

### 反转悬念型

`很多人以为【旧判断】，没想到【新进展】出现后，真正的问题才露出来`

适合：舆论反转、事件后续、人物评价变化、争议复盘。

### 强情绪 / 疑问型

`【事件】为什么会引发这么大争议？关键可能不在【表面】，而在【深层原因】`

适合：社会热点、娱乐争议、民生事件、消费纠纷。

### 普通人提醒型

`【新规/变化/事件】来了，这几类人要提前注意，别等【后果】才知道`

适合：政策、出行、消费、教育、健康生活、平台规则。

### 表面现象 + 真正问题

`【事件】看着是【表面】，真正的问题其实是【深层矛盾】`

适合：社会争议、家庭矛盾、职场冲突、消费纠纷、健康习惯。

### 争议提问型

`【事件】到底是谁的责任？看完【细节】，很多人可能会改变态度`

适合：责任边界、赔偿、物业、学校、职场、服务纠纷。

### 数字清单型

`【事件】背后，这3个细节值得普通人注意`

适合：生活提醒、复盘、规则解释、避坑。

### 反常识型

`很多人以为【常识判断】，但【事件】说明，真正该担心的是【新判断】`

适合：健康、消费、教育、职场、家庭关系。

### 事后三点型

`【事件】结束后，真正值得看的不是【结果】，而是这3个变化`

适合：赛后、争议处理后、政策落地后、舆论反转后。

### 对立双方型

`【A方】觉得委屈，【B方】也有苦衷，这件事最难的是【矛盾】`

适合：家校、物业业主、消费者商家、员工公司、司机乘客。

### 时间节点 + 冲突结果

`【时间】刚发生，【人物/机构】就【动作】，这件事最值得注意的不是【表面】，而是【核心矛盾】`

适合：连续进展、快速反转、事件升级。

### 强情绪 + 解释型

`太意外了！【事件】之后，【群体/人物】最该补上的其实是【认知】`

适合：意外事件、生活安全、公众认知误区。慎用，避免夸张。

### 清单总结型

`【事件】背后，有【数字】个变化值得注意，尤其是最后一点`

适合：复杂事件拆解、政策影响、行业变化、体育赛后、热点复盘。

### 普通人利益型

`【事件】和普通人有什么关系？真正要看的不是热闹，而是【具体影响】`

适合：政策、消费、就业、教育、平台规则、社会民生。

## 赛道适配

### 国际 / 军事

- `【时间】之后，【国家/机构】突然【动作】，这件事的关键不是【表面】，而是【核心变化】`
- `【人物/机构】刚表态，【另一方】就有动作，真正值得注意的是这3个信号`
- `很多人只看到【冲突表面】，但真正影响局势的，可能是【深层因素】`

### 体育

- `【比赛/事件】结束后，真正该看的不是比分，而是这3个信号`
- `【球员/队伍】没想到，【结果】之后，争议反而更大了`
- `【数字】分也救不了【队伍/球员】，问题可能出在这3点`

### 娱乐

- `【明星/作品】风波之后，真正被看见的不是热闹，而是【问题】`
- `【人物】没想到，最尴尬的不是【表面】，而是【反转】`

### 健康生活

- `【事件】别急着怪【表面原因】，这3个习惯更值得注意`
- `很多人以为【常识】，但【事件】提醒：真正要改的是【习惯】`

### 商业/产品/消费

- `【产品/公司】这次变化，最该注意的不是【价格/功能】，而是【影响】`
- `【新规/调整】来了，这几类用户要提前看清楚`

### 教育/职场

- `【事件】之后，真正该反思的不是谁赢了，而是【边界/规则】`
- `【A方】觉得委屈，【B方】也有压力，问题卡在【矛盾】`

### 科技 / 数码

- `【产品/技术】来了，真正改变的不是【功能】，而是【使用门槛】`
- `很多人以为【旧认知】，但【新技术】说明，普通人也要重新看懂这件事`
- `【公司/产品】这次更新，最值得注意的不是参数，而是它会影响谁`

### 财经 / 楼市

- `【政策/市场变化】之后，真正该看的不是涨跌，而是这3个信号`
- `【事件】看着是市场波动，其实背后是【深层变化】`
- `普通人别只看【价格/数据】，这次变化真正影响的是【人群/利益】`

### 社会 / 民生

- `【事件】看着只是个案，其实提醒了很多人一个现实问题`
- `【人物/群体】为什么会争议这么大？真正的矛盾可能在这里`
- `【事件】之后，普通人最该注意的不是热闹，而是这条边界`

## 禁用写法

| 不要这样写 | 原因 | 替代方向 |
|---|---|---|
| 震惊！内幕曝光，太可怕了 | 标题党、容易违规 | 写具体事实和普通人影响 |
| 某某必有大事发生 | 无依据推断 | 写“这3个信号值得关注” |
| 医生都不敢说的秘密 | 健康谣言风险 | 引用公开观点，只做生活提醒 |
| 全网炸了，真相来了 | 泛化夸张 | 写清楚争议点 |
| 某地人都如何如何 | 地域攻击风险 | 聚焦事件，不扩大到群体 |

## 默认生成格式

```text
1. 标题
   类型：普通人提醒型
   结构：事件 + 普通人利益 + 风险提醒
   适用场景：政策、消费、民生、健康生活
   风险提示：如涉及事实变化，发布前必须核查最新来源

2. 标题
   类型：数字具体化型
   结构：事件 + 3个细节/信号/变化
   适用场景：热点复盘、复杂事件拆解
   风险提示：数字必须有来源，不能凭空编造

3. 标题
   类型：反转悬念型
   结构：旧判断 + 新进展 + 深层问题
   适用场景：舆论反转、后续进展、争议事件
   风险提示：不能把推断写成事实
```

## 生成要求

1. 每次优先生成 10 到 20 个标题，除非用户指定数量。
2. 每个标题尽量控制在 18 到 32 个汉字。
3. 标题必须保留事实核心，不得编造人物、数字、结果。
4. 涉及当天热点、政策、健康、法律、金融、灾害时，必须提示先核查事实。
5. 如果信息不足，使用“可能是”“这是推断”，不能写成确定结论。
6. 不要承诺“爆款”“必火”“全网第一”等无法证明的效果。

## 输出格式

```text
1. 【标题】
   类型：【模板类型】
   适合角度：【写作方向】
   风险提醒：【需要核查或避免夸大的点】

2. 【标题】
   类型：【模板类型】
   适合角度：【写作方向】
   风险提醒：【需要核查或避免夸大的点】
```
